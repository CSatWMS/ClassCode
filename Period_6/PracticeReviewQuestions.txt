int a = 0;
int b = 10;
int c = 0;

while (a < 5){
	print(c + ", ");
	if (a % 2 == 0){
		print(b);
	}
	a++;
	b--;
	c += 3;
	println();
}
print(a + ", " + b + ", " + c);
------------------------------------------
Given start and stop, write a function to inclusively move up from start to stop, printing certain numbers. Only print numbers between that are divisible by 3 and odd. You can assume start is less than stop.
